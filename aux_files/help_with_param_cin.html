CNApp applies a re-segmentation approach aiming at correcting potential background noise and amplitude divergence due to technical variability. Parameters for re-segmentation are:

  • Minimum segment length [100000 bp]:
    minimum length of a copy-number segment to be included in the analysis. Segments below this value will be automatically discarded
  • Maximum distance between segments [1000000 bp]:
    maximum distance allowed between segments to be merged. Segments above this value will not be merged even accomplishing the remaining required parameters
  • Maximum amplitude (seg.mean) deviation between segments [0.16]:
    maximum difference allowed in seg.mean values between adjacent segments
  • Minimum amplitude (seg.mean) deviation from segment to zero [0.16]:
    segments must have seg.mean values higher than 0.16 or lower than -0.16 to be considered for re-segmentation, otherwise, they will be re-segmented to 0 baseline.
  • Maximum BAF deviation between segments [0.1]:
    maximum deviation allowed in BAF values between adjacent segments
Users can customize these parameters (Advanced option) or use the default ones.

Re-segmentation step can be skipped, in wich case CNA scores will be computed considering segments from the input file.




CNA Scores (i.e. Broad, Focal and Global [BCS, FCS and GCS, respectively]) are computed through copy number events classification and weigth, according to customizable CNA level thresholds (see our manuscript Franch-Exp√≥sito et al., 2018 for mor details).

Default thresholds for CNA calling are:

CNA level log2ratio values Number of copies
High-level gain 1 >= 4
Medium-level gain 0.58 [3 - 4)
Low-level gain 0.2 [2.3 - 3]
Low-level loss -0.2 (1 - 1.7]
Medium-level loss -1 (0.6 - 1]
High-level loss -1.74 <= 0.6



CNAs classification is performed by applying whole-chromosome and chromosome arm coverage cutoffs. Default cutoffs are as follows and can be changed by the user:

  • Minimal coverage for chromosomal CNAs [0.9]:
    considering the relative length of a segment to the whole-chromosome, the segment will be tagged as chromosomal if covers more than 90% of the chromosome
  • Minimal coverage for arm-level events [0.5]:
    considering the relative length of a segment to a chromosome-arm, the segment will be tagged as arm-level if covers more than 50% of the arm
