The genomic region profiling generates genome-wide profiles by transforming using segmented data into genome region profiles defined by a user-selected windows (i.e., chromosome arms, half-arms, cytobands, sub-cytobands or 40-1 Mb windows). Length-relative means are computed for each window by considering amplitude values from those segments included in the specific window (see our manuscript Franch-Exp√≥sito et al., 2018 for mor details).




When Re-Seg & Score section has been run, options to use resources from that sections are available:

  • Use new segments:
    Re-segmented data (or not, if skipped) can be selected for analysis in this section
  • Choose CNAs according to their classification:
    taking advantage of CNA classification from Re-Seg & Score, all, only broad, or only focal events can be selected (i.e. recommended to selected focal events if the aim of study are focal alterations regions and frequencies)
  • Add CNA scores as annotation variables:
    CNA Scores (i.e. BCS, FCS and GCS [see Re-Seg & Score section]) can be loaded as annotation tracks in this section
