CNApp shiny web-tool works with copy number segmented data from multiple platforms such as SNP arrays, aCGH and next generation sequencing, including whole-exome, whole-genome and targeted sequencing.

Minimum input file must be five columns matrix with ' ID ', ' chr ', ' loc.start ', ' loc.end ' and ' seg.mean ' column headers.

Extra columns will be read as annotation variables and can be used to order samples and, more importantly, to perform integrative analysis and association studies with copy number alterations and CNA Scores computed by CNApp.

Genome builds hg19 and hg38 are accepted and must be specified.

Decimal character ('.' or ',') and column field ('tab' or ',') must be specified.
